

# 📄 PDF Document Manager

Eine einfache, übersichtliche App zum Speichern, Organisieren und Betrachten deiner PDF-Dokumente – direkt im Browser, ohne Account, auf Smartphone und PC.

---

## 1. Startseite / Dokumenten-Bibliothek
- Übersichtliche Kachelansicht oder Listenansicht aller gespeicherten PDFs
- Upload-Button zum Hinzufügen neuer PDFs (Drag & Drop auf Desktop, Dateiauswahl auf Mobile)
- Suchleiste zum schnellen Finden von Dokumenten nach Name oder Tag
- Sortierung nach Name, Datum, zuletzt geöffnet
- Umschalten zwischen Kachel- und Listenansicht

## 2. Ordner & Tags
- Ordner erstellen, umbenennen und löschen
- PDFs per Drag & Drop oder Menü in Ordner verschieben
- Tags vergeben und nach Tags filtern (farbige Labels)
- Sidebar-Navigation mit Ordnerstruktur und Tag-Filter

## 3. PDF-Viewer
- Integrierter PDF-Viewer zum Betrachten der Dokumente direkt in der App
- Seitennavigation (vor/zurück, Seitenummer-Eingabe)
- Zoom-Steuerung (rein/raus, an Breite anpassen)
- Textsuche innerhalb des PDFs mit Hervorhebung der Treffer
- Lesezeichen setzen für einzelne Seiten, die dann in einer Seitenleiste aufgelistet werden

## 4. Lokale Speicherung
- Alle Dokumente und Metadaten (Ordner, Tags, Lesezeichen) werden im Browser gespeichert (IndexedDB)
- Kein Account oder Server nötig – alles bleibt auf dem Gerät
- Hinweis: Daten gehen verloren, wenn Browserdaten gelöscht werden

## 5. Dark/Light Mode
- Umschaltbar über einen Toggle in der oberen Leiste
- Systemeinstellung wird automatisch erkannt als Standard

## 6. Responsive Design
- Optimiert für Smartphone und Desktop
- Auf Mobile: Bottom-Navigation, Touch-freundliche Steuerung im PDF-Viewer
- Auf Desktop: Sidebar-Navigation, Tastaturkürzel für Navigation
